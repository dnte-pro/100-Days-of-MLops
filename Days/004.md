# Creating a Standard ML Project Structure

## Your task is to substitute all occurrences of the string Random with LUSV within the XML file located at /root/nautilus.xml on the jump host server.

On the jumphost server run:
```bash
sed -i 's/Random/LUSV/g' /root/nautilus.xml
```
The command edits the file and replaces all "Random" to "LUSV" in the file 

### Verifying the changes
```bash
grep LUSV /root/nautilus.xml
```
The output shows all the LUSV that replaces the Random in the file
